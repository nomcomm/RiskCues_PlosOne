# Analysis for Schmälzle, Hartung, Imhof, Kenter, Renner, & Schupp (submitted)

Data and analysis code for *Schmälzle, R., Hartung, F-M., Imhof, M., Kenter, A., Renner, B. & Schupp, H.T. (submitted). Risky looks?
